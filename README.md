食用指南

--> 下载StellarisModManager并在任意处解压

--> 点击tellarisModManager.exe运行

--> 点击左上角Settings更改语言（如果能看懂还是别改了吧， GPT师傅翻译的，非常抽象）

--> 点击左上角创意工坊->浏览创意工坊进入内置浏览器

--> 点击Mod缩略图上的蓝色加号将其放入下载队列

--> 第一次下载之前会要求你设置你的mod要下载到哪个目录（设置到随便哪个能让你的Irony能读取到的地方）

--> 第一次下载会花费较长时间，因为要setup SteamCMD

--> 下载完了点开Irony确认一下
